</script>

  <hr-tooltip
    trigger="click"
    content="click to display tooltip"
    placement="bottom"
  >
    <hr-button
      >click to display tooltip</hr-button
    >
  </hr-tooltip>
</template>

更多内容的文字提示 ​

展示多行文本或者设置内容的格式 通过 content 插槽设置内容

<template>
  <hr-tooltip placement="top">
    <template #content> multiple lines<br />second line </template>
    <hr-button>Top center</hr-button>
  </hr-tooltip>
</template>

禁用状态 ​

禁用状态，鼠标悬停无法触发文字提示。disabled 属性可以满足这个需求。

<script lang="ts" setup>
import { ref } from "vue";

const disabled = ref(false);
</script>

<template>
  <hr-tooltip
    :disabled="disabled"
    content="click to close tooltip function"
    placement="bottom"
  >
    <hr-button @click="disabled = !disabled"
      >click to {{ disabled ? "active" : "close" }} tooltip function</hr-button
    >
  </hr-tooltip>
</template>
